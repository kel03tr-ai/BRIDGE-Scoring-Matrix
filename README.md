# BRIDGE Scoring Matrix — Interactive Tool

An interactive scoring matrix for the AIETP 2026 training programme, designed for facilitators to project during live sessions. Participants compare general-purpose LLMs with education-specific AI tools using the BRIDGE framework.

## What It Does

During the Tool Tasting activity in Session 2, groups of teachers test two AI tools side by side — one general-purpose LLM (ChatGPT, Claude, Gemini, or DeepSeek) and one education-specific tool (MagicSchool AI, Eduaide.ai, Curipod, SchoolAI Sidekick, Teachermatic, Teachy, or TeachMateAI). Each group rates both tools on the six BRIDGE criteria (Building language skills, Reduce workload, Intuitive to use, Developing learning processes, Get students together, Extend learning) using a 1–5 scale.

This tool lets the facilitator enter scores live as groups report out. Totals update instantly, and visual charts make the comparison immediately clear to everyone in the room.

## Features

- Pre-loaded with 7 K-12 Turkish EFL scenarios (High School, Secondary School, Primary School) and tool pairings
- Auto-calculated totals with star indicator for the higher-scoring tool in each group
- Summary bar chart comparing all groups at a glance
- Per-group detail charts showing each BRIDGE criterion side by side
- Fullscreen mode for projection
- Score validation (1–5 only)
- Reset button to clear all data
- Single self-contained HTML file with no external dependencies

## BRIDGE Framework

| Criterion | Question |
|-----------|----------|
| **B** — Building language skills | Does it support language learning goals? |
| **R** — Reduce workload | Does it save you meaningful time? |
| **I** — Intuitive to use | Can you and your students use it easily? |
| **D** — Developing learning processes | Does it support sound pedagogy? |
| **G** — Get students together | Does it enable collaboration? |
| **E** — Extend learning | Does it support learning beyond class time? |

## How to Use

1. Open `index.html` in any modern browser (or visit the GitHub Pages URL)
2. Click **Fullscreen** to project for the whole room
3. As each group shares their scores, enter them into the table
4. Charts update automatically — use them to guide discussion
5. Use **Reset All** to clear scores between sessions
